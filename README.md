# 電子パーツ在庫ビューア

個人所有の電子部品のデータベースとビューアです。

以下のような状況がよく起こるので、スマホなどで簡単に確認できるようにしてみました。

- 手持ちの部品で設計したいけど、何があったっけ？
- 予定外にパーツ屋に来たけど、在庫切らしてた部品はなんだっけ？
- 店頭でお買い得品を見つけたけど、これ持ってなかったけ？

パーツの管理はSQLiteで行い、ビューアをGitHub Pagesでホスティングしています。

パーツ情報の検索。表示機能のみをサポートします。更新機能はありません。
パーツ情報の更新は、[eparts.db](./eparts.db)を更新、リポジトリにコミットすることで行います。

# 環境設定

## リポジトリの設定

1. [schema.sql](./schema.sql)がデータベースのスキーマです。この形式で`eparts.db`を生成します。
2. リポジトリに、`eparts.db`と`index.html`を登録します。

## GitHub Pagesのホスティング設定

1. GitHubのリポジトリページに移動  
  「Settings」タブをクリック
2. 左側のサイドバーから「Pages」を選択  
  「Source」セクションで、ブランチを選択（通常は「main」または「master」）
3. ルートディレクトリ「/(root)」を選択  
  「Save」をクリック

## パーツの登録

`eparts.db`にパーツのデータを登録します。
